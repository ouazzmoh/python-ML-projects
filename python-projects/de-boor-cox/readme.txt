Ce script permet de tracer des courbes à l'aide des B-Splines:
1. Utilisation de l'algorithme de deBoor-Cox pour tracer des points de la courbe
2. Utilisation du principe de subdivision pour tracer la courbe avec une convergence rapide (En rouge)
3. Saturer les noeuds pour avoir le polygone Bézier associé a la courbe (En bleu clair)
